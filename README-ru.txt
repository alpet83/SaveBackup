Установка для игры "Метро Исход"

Чтобы воспользоваться программой:
1. Загрузите архив svbackup_win32, а так-же файлы svbackup.conf и metro_exodus.lua. 
2. Распакуйте программу из архива, в одну папку с этими файлами, сделайте при необходимости ярлык на "Рабочий стол"

После запуска потребуется скорректировать пути к папкам где лежат сохраненки, и куда будут выкладыватся копии.
